# CN322 Network Security - Web Application Firewall Project


โครงงานนี้มีวัตถุประสงค์เพื่อพัฒนา **ระบบ Web Application Firewall (WAF)**  
ซึ่งทำหน้าที่เป็นตัวกลางระหว่างผู้ใช้งาน (Client) และเว็บแอปพลิเคชันเป้าหมาย (Target Web Application)  
เพื่อช่วยตรวจสอบและป้องกันการโจมตีที่พบบ่อยในระดับเว็บแอปพลิเคชัน ได้แก่

- **SQL Injection (SQLi)**
- **Cross-Site Scripting (XSS)**

ระบบ WAF ที่พัฒนาขึ้นใช้หลักการ **Signature-Based Detection**  
โดยอาศัยการตรวจจับรูปแบบของการโจมตี (Attack Patterns) ด้วย Regular Expression (Regex)  
หากตรวจพบคำขอที่มีลักษณะเข้าข่ายการโจมตี ระบบจะทำการ **บล็อกคำขอทันที**  
พร้อมทั้งบันทึกข้อมูลการโจมตีลงในฐานข้อมูล เพื่อใช้ในการตรวจสอบย้อนหลัง

### ความสามารถหลักของระบบ
- ตรวจสอบ HTTP/HTTPS Request จากผู้ใช้งาน
- ตรวจจับ SQL Injection จากคำสั่งหรือรูปแบบที่เป็นอันตราย
- ตรวจจับ XSS จาก Script, Event Handler และ Tag ที่มีความเสี่ยง
- บันทึก Log การโจมตี (IP, ประเภทการโจมตี, Payload, Path และเวลา)
- อนุญาตให้คำขอที่ปลอดภัยส่งต่อไปยัง Target Web Application ได้ตามปกติ

ระบบถูกพัฒนาโดยใช้ภาษา **Python** และ **Flask Framework**
